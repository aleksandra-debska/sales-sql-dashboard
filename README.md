# superstore-sales-analysis

## Project Overview
This project focuses on extracting Key Performance Indicators (KPIs) using SQL and visualizing them through a Power BI dashboard. The goal is to identify revenue drivers, customer segments, and sales growth patterns.

## Tools & Technologies
* Language: SQL (Oracle SQL)
* Visualization: Power BI
* Functionalities: Data validation, trend analysis, and KPI tracking

## Key Insights
* Top Category: Technology leads with 0.83 mln in sales.
* Customer Segment: The Consumer segment generates the highest share of revenue (50.76%).
* Regional Performance: The West region is the top-performing market at 0.71 mln.
* Product Performance: The Canon imageCLASS 2200 Advanced Copier is the top-selling product, contributing significantly to overall sales.
* Growth Trend: Annual sales show a consistent upward trajectory, peaking at 0.72 mln in 2018.

## Key Performance Indicators (KPIs)
* Total Sales: 2.26 mln.
* Total Orders: 9.8k.
* Avg. Order Value: 230.77.

## Files in Repository
* sales.sql - SQL script for data validation, revenue calculation, and performance tracking.
* Superstore_Sales_Dashboard.pbix - Interactive Power BI dashboard file.
* retail_sales_data.xlsx - Dataset used for the analysis.
